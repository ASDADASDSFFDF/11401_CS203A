## 堆疊（Stack）
### 一、定義
#### 堆疊（Stack）是一種線性資料結構，遵循 LIFO（Last In, First Out，後進先出）
### 二、基本概念
#### 只能在頂端（Top)進行操作
#### 主要操作：
- Push：加入元素
- Pop：移除元素
- Peek / Top：查看頂端元素
### 三、特性
- 無法隨機存取
- 只能存取頂端元素
- 操作簡單、效率高
### 四、優缺點
#### 優點
- Push / Pop 時間複雜度為 O(1)
- 適合暫存資料
#### 缺點
- 只能從 Top 操作
- 不適合搜尋大量資料
### 五、時間複雜度
```
操作	時間複雜度
Push	O(1)
Pop 	O(1)
Peek	O(1)
Search	O(n)
```
