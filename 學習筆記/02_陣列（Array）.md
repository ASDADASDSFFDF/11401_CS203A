### 一、陣列的定義
#### 陣列（Array）是一種線性資料結構，用來儲存相同資料型別的一組元素，且元素在記憶體中是連續配置。
### 二、陣列的基本特性
#### 1.資料型別需一致
#### 2.佔用連續的記憶體空間
#### 3.索引值從 0 開始編號
#### 4.存取速度快（O(1)）
### 三、陣列的優缺點
#### 優點:
- 存取速度快（O(1)）
- 結構簡單、容易實作
- 適合固定大小的資料儲存
#### 缺點:
- 大小固定
- 插入與刪除成本高 O(n)
- 無法直接儲存不同型別資料
### 四、時間複雜度分析
```
操作	時間複雜度
存取	O(1)
搜尋	O(n)
插入	O(n)
刪除	O(n)
```
