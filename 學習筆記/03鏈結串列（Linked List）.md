# 鏈結串列（Linked List）
## 一、定義
### 鏈結串列（Linked List）是一種線性資料結構，由多個節點（Node)組成。每個節點包含：
### 1.資料（Data）
### 2.指向下一個節點的指標（Pointer）
### 節點在記憶體中不必連續配置
## 二、特性
- 不需要連續記憶體
- 大小可動態改變
- 必須從 Head 依序走訪（無法直接索引）
## 三、優缺點
### 優點
- 可動態新增、刪除節點
- 若已知位置插入與刪除效率高O(1)
- 不浪費記憶體空間
### 缺點
- 無法隨機存取（存取為 O(n)）
- 需要指標空間
- 不適合頻繁搜尋
## 四、Linked List vs Array
```
	       Array	Linked List
記憶體配置	連續	   不連續
大小	    固定	    動態
存取	    O(1)	    O(n)
插入刪除 	O(n)	    O(1)*

*（已知節點位置時）
```
